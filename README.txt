In order to run this code, place the worm.py file in the /tmp/ 
directory on the host machine. Open up terminal and type cd /tmp
Once your terminal is directed to that folder, run the worm file
by using the command "python worm.py"

Extra credit Cleaning function was implemented in this worm.py file.
In order to run the cleaning function type "python worm.py -c" or "python worm.py --clean"
